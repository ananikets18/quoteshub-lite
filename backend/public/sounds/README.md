# Notification Sounds

## Setup

Place your notification sound file here as `notification.mp3`.

## Recommended Sources

1. **Notification Sounds** - https://notificationsounds.com/
2. **Freesound** - https://freesound.org/
3. **Zapsplat** - https://www.zapsplat.com/

## Requirements

- Format: MP3
- Duration: 0.5-2 seconds recommended
- File size: < 100KB recommended
- Volume: Normalized to -3dB

## Example Sound

For testing, you can use a simple beep sound or download from the sources above.

## Note

The notification sound is optional. If the file doesn't exist, the notification will still work without sound.
